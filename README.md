# How to get started

1. Install latest version of Node.js

2. Install Expo CLI and yarn package.
   ```Shell
   > npm install --global expo-cli
   > npm install --global yarn
   ```

3. Install all dependencies for the project.
   ```Shell
   > yarn install
   ```

4. Run with Expo CLI.
   ```Shell
   > expo login
   > expo start [--tunnel]
   ```
